The Connection Between Minimizing Sum-of-squares and Maximizing Likelihood

Posted on 2018-11-22 |

This post rephrases section 1.2.5 of [1] that shows the connection between minimizing sum-of-squares error function and maximizing likelihood in curve fitting.

Problem definition

The goal in the curve fitting problem is to be able to make predictions for the target variable $T$ given some new value of the input variable $X$ on the basis of a set of training data comprising $N$ input values $\mathbf{X} = \left[ x_{1}, \dots, x_{1} \right]^{T}$ and their corresponding target values $\mathbf{T} = \left[ t_{1}, \dots, t_{N} \right]^{T}$.

Probabilistic perspective solution

We can express our uncertainty over the value of the target variable using a probability distribution.

For this purpose, we shall assume that, given the input value $x$, the corresponding target value $t$ has a Gaussian distribution with a mean equal to the value $y(x, \mathbf{w}) = \sum^{M}_{k=0} w_{k} x^{k}$, where $M$ is the order of the polynomial function $y$. Under this assumption there is

\begin{align}
p(t \vert x, \mathbf{w}, \beta ) = \mathcal{N}\left( t \vert y(x, \mathbf{w}), \beta^{-1} \right), \tag{1}
\label{eq:noise_dist}
\end{align}

where the precision parameter $\beta$ corresponds to the inverse variance of the distribution.

Then we use the training data to determine the values of the unknown parameters $\mathbf{w}$ and $\beta$ by maximum likelihood. If the data are assumed to be drawn independently from the distribution \eqref{eq:noise_dist}, then the likelihood function is given by

\begin{align}
p(\mathbf{T} \vert \mathbf{X}, \mathbf{w}, \beta ) = \prod^{N}_{n=1} \mathcal{N}\left( t_{n} \vert y(x_{n}, \mathbf{w}), \beta^{-1} \right). \tag{2}
\label{eq:lik}
\end{align}

Substituting for the form of the Gaussian distribution, we obtain the log likelihood function in the form

\begin{align}
\ln p(\mathbf{T} \vert \mathbf{X}, \mathbf{w}, \beta ) = - \frac{\beta}{2} \sum^{N}_{n=1} \left[ y(x_{n}, \mathbf{w}) - t_{n} \right]^{2} + \frac{N}{2} \ln \beta - \frac{N}{2} \ln 2 \pi. \tag{3}
\label{eq:log-lik}
\end{align}

Consider first the determination of the maximum likelihood solution for the polynomial coefficients, which will be denoted by $\mathbf{w}_{ML}$. These are determined by maximizing \eqref{eq:log-lik} with respect to $\mathbf{w}$. For this purpose,

  • we can omit the last two terms on the right-hand side of \eqref{eq:log-lik} because they do not depend on $\mathbf{w}$;

  • we can replace the coefficient $\beta / 2$ with $1 / 2$ since scaling the log likelihood by a positive constant coefficient does not alter the location of the maximum with respect to $\mathbf{w}$;

  • we can equivalently minimize the negative log likelihood instead of maximizing the log likelihood.

As a result, we see that maximizing likelihood is equivalent, so far as determining $\mathbf{w}$ is concerned, to minimizing the sum-of-squares error function $\sum^{N}_{n=1} \left[ y(x_{n}, \mathbf{w}) - t_{n} \right]^{2}$. Thus the sum-of-squares error function has arisen as a consequence of maximizing likelihood under the assumption of a Gaussian noise distribution.
