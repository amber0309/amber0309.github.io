Kernel Embedding-Based Clustering

Posted on 2017-06-07 |

This post is an introduction to my clustering method called Kernel Embedding-Based Clustering (KEBC).

Problem Description

KEBC aims to cluster $N$ data groups into $C$ clusters according to the conditional distribution of each group.

Specifically, assume we are given a set $\mathbf{G}$ which consists of $N$ data groups, i.e. $\mathbf{G} = \lbrace G_{1}, \dots, G_{N} \rbrace $ and each group consists of bi-variate tuples, e.g. $ G_{i} = \lbrace (x^{(i)}_{1}, y^{(i)}_{1} ), \dots, (x^{(i)}_{n_{i}}, y^{(i)}_{n_{i}} ) \rbrace $ ($n_{i}$ denotes the number of points in group $i$), the goal of KEBC is to find the clustering such that groups within each cluster follow the same or very similar conditional distributions.

Formally, let $d\left( P^{(i)}_{Y|X}, P^{(j)}_{Y|X} \right) $ be a dissimilarity measure between conditional distributions, where $P^{(i)}_{Y|X}$ denotes the conditional distribution of $Y$ on $X$ of group $i$. Our objective is to find a $C$-partitioning of the $N$ groups s.t., for $\mu_{S_{j}}$ being the mean conditional distribution representation of cluster $S_{j}$,
\begin{aligned} \mathbf{S}^{*} = arg\max_{\mathbf{S}} \sum_{i=1}^{N} \sum_{P^{(i)}_{Y|X} \in S_{j}} d^{2}\left( P^{(i)}_{Y|X}, \mu_{S_{j}} \right) \end{aligned}

Scope of Application

KEBC could be applied in any problem that has a (conditional) distribution-distinguishable interpretation.
